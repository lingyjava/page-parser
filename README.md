# Page Parser - 网页解析Chrome插件

一个强大的Chrome浏览器插件，用于解析网页内容，支持为不同网站配置自定义的解析规则。

## 🚀 安装使用

### 1. 加载插件

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `packages/chrome` 目录

### 2. 配置解析规则

1. 点击插件图标打开popup
2. 点击"配置规则"按钮打开配置页面
3. 点击"添加新配置"
4. 输入网站域名（例如：www.example.com）
5. 添加解析字段和对应的CSS选择器
6. 保存配置

### 3. 使用插件

1. 访问已配置的网站
2. 点击插件图标
3. 查看解析预览
4. 选择"发送到接口"或"保存为JSON"
